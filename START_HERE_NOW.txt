                    🚀 এখনই শুরু করুন - 3টি ধাপ:
═══════════════════════════════════════════════════════════════════════════

ধাপ 1️⃣  BACKEND চালু করুন (নতুন Terminal):
────────────────────────────────────────────────────────────

  1. কমান্ড প্রম্পট বা PowerShell খুলুন
  2. নিচের কমান্ড চালান:

     cd C:\Users\User\social_chat_app\backend
     python -m venv venv
     venv\Scripts\activate
     pip install -r requirements.txt
     copy .env.example .env
     python app.py

  ✅ সাফল্য: "Running on http://127.0.0.1:5000"

──────────────────────────────────────────────────────────────

ধাপ 2️⃣  FRONTEND চালু করুন (নতুন Terminal):
────────────────────────────────────────────────────────────

  1. নতুন Terminal খুলুন
  2. নিচের কমান্ড চালান:

     cd C:\Users\User\social_chat_app\frontend
     npm install
     copy .env.example .env
     npm start

  ✅ সাফল্য: "Compiled successfully!" এবং 
              http://localhost:3000 এ ওপেন হবে

──────────────────────────────────────────────────────────────

ধাপ 3️⃣  FIREBASE কনফিগার করুন:
────────────────────────────────────────────────────────────

  1. https://console.firebase.google.com/ এ যান
  2. নতুন প্রজেক্ট তৈরি করুন (নাম: social-chat-app)
  3. Authentication এ যান:
     • Email/Password Enable করুন
     • Google Sign-In Enable করুন
     • Facebook Sign-In Enable করুন
  4. Project Settings এ যান:
     • Service Account Private Key ডাউনলোড করুন
     • Web SDK config কপি করুন
  5. .env ফাইল আপডেট করুন:
     
     backend/.env এ:
       FIREBASE_API_KEY=...
       FIREBASE_PROJECT_ID=...
       etc.
     
     frontend/.env এ:
       REACT_APP_FIREBASE_API_KEY=...
       REACT_APP_FIREBASE_PROJECT_ID=...
       etc.
  
  6. উভয় সার্ভার রিস্টার্ট করুন (Ctrl+C এবং আবার চালান)

  ✅ সম্পূর্ণ!

                         🔗 API এন্ডপয়েন্ট:
═══════════════════════════════════════════════════════════════════════════

  ✅ POST /api/auth/signup
     ইনপুট: নাম, ইমেইল, পাসওয়ার্ড, etc.
     আউটপুট: নতুন ইউজার + Token

  ✅ POST /api/auth/login
     ইনপুট: ইমেইল/ফোন, পাসওয়ার্ড
     আউটপুট: Token + ইউজার তথ্য

  ✅ POST /api/auth/forgot-password
     ইনপুট: ইমেইল, নাম, জন্মদিন
     আউটপুট: সাপোর্ট কন্টাক্ট

  ✅ POST /api/auth/google
     ইনপুট: Google Token
     আউটপুট: JWT Token

  ✅ POST /api/auth/facebook
     ইনপুট: Facebook Token
     আউটপুট: JWT Token

  ✅ GET /api/health
     সার্ভার স্ট্যাটাস চেক

                         🎨 UI পেজ (প্রস্তুত):

  🔐 LOGIN PAGE (http://localhost:3000/login)
     • সুন্দর সায়ান-বেগুনি ডিজাইন
     • Email/Phone ইনপুট
     • পাসওয়ার্ড (show/hide অপশন)
     • "Forgot Password" লিংক

     • Google & Facebook বাটন

     • 3 মেথড: Manual, Google, Facebook
     • Terms & Conditions মোডাল
     • ফর্ম ফিল্ড:
       - Full Name (প্রয়োজনীয়)
       - Date of Birth (প্রয়োজনীয়)
       - Gender (পুরুষ/মহিলা/অন্যান্য)
       - Email (প্রয়োজনীয়)
       - Phone (ঐচ্ছিক)
       - Password (প্রয়োজনীয়, ৬+ অক্ষর)
       - Confirm Password (প্রয়োজনীয়)
     • ভ্যালিডেশন + Error মেসেজ
     • Success নোটিফিকেশন

  🔑 FORGOT PASSWORD PAGE (http://localhost:3000/forgot-password)
     • Email/Phone সার্চ
     • Name ও DOB ভেরিফিকেশন
     • সাপোর্ট কন্টাক্ট তথ্য প্রদর্শন
     • WhatsApp লিংক

  🏠 HOME PAGE (http://localhost:3000/home)
     • সুরক্ষিত রুট (লগইন প্রয়োজন)

                       🆘 সমস্যা সমাধান:
═══════════════════════════════════════════════════════════════════════════

  Q: "python is not recognized"
  A: Python ইনস্টল করুন বা সম্পূর্ণ পাথ ব্যবহার করুন
     C:\Users\User\AppData\Local\Programs\Python\Python314\python.exe

  Q: "npm: command not found"
  A: Node.js ডাউনলোড এবং ইনস্টল করুন:
     https://nodejs.org/

  Q: "Port 3000/5000 already in use"
  A: অন্য PORT ব্যবহার করুন:
     Backend: app.run(port=5001)
     Frontend: PORT=3001 npm start

  Q: "Firebase auth failed"
  A: .env ফাইল সঠিকভাবে পূরণ করা হয়েছে কিনা চেক করুন

  Q: "CORS error"
  A: Backend এ CORS Enable করা আছে, নিশ্চিত করুন
